Tu tracker financiero personal — PWA instalable en Android, iPhone y PC.

---

```
    └── icon.svg    ← Ícono de la app
```

## 🚀 Cómo publicar en GitHub Pages (gratis, 5 minutos)

### Paso 1 — Crea una cuenta en GitHub
Ve a https://github.com y crea tu cuenta si no tienes una.

### Paso 2 — Crea un repositorio nuevo
1. Clic en el botón verde **"New"** (o en **+** → New repository)
2. Nombre del repo: `finanzas-app` (o el que quieras)
3. Ponlo en **Public**
4. Clic en **"Create repository"**

### Paso 3 — Sube los archivos
En la página del repo vacío, clic en **"uploading an existing file"**:
1. Arrastra TODOS los archivos: `index.html`, `style.css`, `app.js`, `manifest.json`, `sw.js`
2. Para el ícono, primero crea la carpeta: en el campo de nombre escribe `icons/icon.svg` — GitHub crea la carpeta automáticamente
3. Clic en **"Commit changes"**

### Paso 4 — Activa GitHub Pages
1. Ve a **Settings** (pestaña del repo)
2. Sidebar izquierdo → **Pages**
3. En "Branch", selecciona **main** → carpeta **/ (root)**
4. Clic en **Save**

### Paso 5 — Tu URL (lista en ~1 minuto)
Tu app estará en:
```
https://TU_USUARIO.github.io/finanzas-app/
```

## 📱 Cómo instalar como App

### Android (Chrome)
1. Abre la URL en Chrome
2. Menú (3 puntos) → **"Añadir a pantalla de inicio"**
3. Confirma → aparece el ícono como app nativa ✅

### iPhone / iPad (Safari)
1. Abre la URL en Safari (importante: debe ser Safari)
2. Botón de compartir (□↑) → **"Añadir a pantalla de inicio"**
3. Confirma → ícono en el home screen ✅

### PC Windows / Mac (Chrome o Edge)
1. Abre la URL en Chrome o Edge
2. En la barra de dirección aparece un ícono de instalar (⊕ o pantalla con flecha)
3. Clic → **"Instalar"**
4. La app aparece como programa independiente en tu escritorio ✅
